# Employee Attendance System

Software solution for managing employee attendance and leave requests in a company environment.

## Features
- employee login system
- attendance tracking
- leave request creation
- leave request approval and modification
- overview of employee requests

## Technologies
- C#
- .NET
- Windows Forms
- Visual Studio
